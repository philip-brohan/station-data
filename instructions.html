How to use this dataset¶

This document is kept under version control in a git repository. The repository is hosted on GitHub (and the documentation made with GitHub Pages). The repository is https://github.com/philip-brohan/station-data.

If you are familiar with GitHub, you already know what to do: If you’d prefer not to bother with that, you can download the whole dataset as a zip file.

The scripts in this package depend on the SEF toolkit which provides support for the SEF data format as a python library.
